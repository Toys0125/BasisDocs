## Il file Basis Encrypted Extension (BEE)

Basis memorizza i contenuti generati dagli utenti come file .BEE che contiene il tuo asset compilato e i metadati ad esso associati. Ogni file .BEE generato viene fornito con una password generata casualmente per accedervi, entrambi trovabili nella directory radice del tuo progetto Unity in: `Basis\Basis\AssetBundles`.

## Evitare GameObject duplicati

- Per impostazione predefinita, il componente BasisAvatar rinominerà tutti i GameObject con nomi duplicati all'interno della gerarchia dell'avatar.

<Callout type="warning">
Se abiliti `Do not auto-rename bones` nelle Opzioni di elaborazione. Assicurati che non ci siano **NESSUN** GameObject con nome duplicato all'interno della gerarchia del tuo avatar, incluso l'armature, poiché questo causerà il fallimento del caricamento dell'avatar dopo la costruzione dell'AssetBundle.
</Callout>

## Costruire l'AssetBundle

- Nella parte inferiore del componente Basis Avatar sul tuo Avatar, verifica che le target di build appropriate per il tuo contenuto siano abilitate. Basis supporta varie piattaforme target, tra cui: Windows, Android, Mac, Linux e iOS.
- Premi il pulsante 'Create Avatar Bee File' e attendi il completamento del processo.

![Crea file BEE Avatar](/img/avatars/7.png)

- I risultati verranno aperti automaticamente in una nuova finestra come mostrato di seguito.

![Risultati della build](/img/avatars/8.png)

- A seconda del tuo hardware, ogni piattaforma potrebbe richiedere in media da uno a due minuti per la prima build. Le build successive dovrebbero richiedere meno tempo grazie alla cache. La costruzione per tutte le piattaforme richiederà probabilmente più tempo e aumenterà le dimensioni complessive del file.

<Callout type="info">
Da sapere:

- Puoi comodamente usare una password personalizzata per il tuo file BEE generato. Questo può semplificare le cose se preferisci una singola password da usare per diversi file BEE.

- Se lo desideri, puoi anche sostituire il nome del file .BEE generato con qualcosa di più facile da ricordare. Tieni presente che riutilizzare lo stesso nome comporterà la sostituzione di qualsiasi versione precedente con quella nuova!


![Crea file BEE Avatar](/img/avatars/7a.png)
</Callout>

<Callout type="info">
Aneddoto del gennaio 2026: Un avatar con 18 materiali che usa Poiyomi URP con 120K poligoni, con dimensioni del pacchetto di 45 MB per piattaforma, ha impiegato 9 minuti e 43 secondi su un Ryzen 7 9800x3D per costruire per tutte le piattaforme target.
</Callout>
