## Jiggles (Optionnel)

**Pour ajouter la physique Jiggle :**

Pour chaque racine jiggle :
- Ajoutez un nouveau composant `Jiggle Rig`, qui peut être placé dans un GameObject séparé, mais il doit se trouver dans la hiérarchie de la racine de votre Avatar.
- Dans le composant Jiggle Rig, assurez-vous de définir un `Root Transform`, qui est un os dans l'armature de l'avatar.
- Si vous avez un os parent (pour plusieurs chaînes d'os enfants auxquels vous souhaitez appliquer le même effet), vous pouvez activer `Motionless Root` pour n'affecter que les os enfants et non la racine elle-même. (Optionnel)
- Ajustez les `Settings` selon vos préférences (activez `Advanced` pour voir tous les paramètres).

**Interaction de collision :**
- Activez `Advanced` pour voir tous les paramètres.
- Activez `Collision` et ajustez `Collision Radius` pour correspondre à l'objet.
