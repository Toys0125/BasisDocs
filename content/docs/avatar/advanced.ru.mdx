## Джиггл (опционально)

**Как добавить физику джиггла:**

Для каждого корня джиггла:
- Добавьте новый компонент `Jiggle Rig`, который можно разместить в отдельном gameObject, но он должен находиться в иерархии корня аватара.
- В компоненте Jiggle Rig обязательно задайте `Root Transform` — кость в арматуре аватара.
- Если у вас есть родительская кость (для нескольких цепочек дочерних костей, к которым вы хотите применить одинаковый эффект), включите `Motionless Root` — тогда эффект будет применяться только к дочерним костям, но не к самому корню. (Опционально)
- Настройте `Settings` по своему вкусу (включите `Advanced` для просмотра всех настроек).

**Взаимодействие при столкновениях:**
- Нажмите `Advanced` для просмотра всех настроек.
- Включите `Collision` и подберите `Collision Radius` под размер объекта.

## Отслеживание глаз и лица (опционально)

- Hai создал подробное руководство по добавлению компонента Automatic Face Tracking к аватару в проекте Basis
- Перейдите по ссылке [здесь](https://docs.hai-vr.dev/docs/basis/avatar-customization/face-tracking) для получения инструкций по его работе.
