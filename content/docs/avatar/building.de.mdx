## Die Basis Encrypted Extension (BEE) Datei

Basis speichert benutzergenerierte Inhalte als .BEE-Datei, die Ihr kompiliertes Asset und dessen Metadaten enthält. Jede generierte .BEE-Datei enthält ein zufällig generiertes Passwort für den Zugriff, das sich beide im Stammverzeichnis Ihres Unity-Projekts befinden: `Basis\Basis\AssetBundles`.

## Doppelte GameObjects vermeiden

- Standardmäßig benennt die BasisAvatar-Komponente alle doppelt benannten GameObjects innerhalb der Hierarchie des Avatars um.

<Callout type="warning">
Wenn Sie `Do not auto-rename bones` in den Verarbeitungsoptionen aktivieren, stellen Sie sicher, dass es **KEINE** doppelt benannten GameObjects in der Hierarchie Ihres Avatars gibt, einschließlich der Armatur, da dies dazu führt, dass Ihr Avatar nach dem Erstellen des AssetBundles nicht geladen werden kann.
</Callout>

## Das AssetBundle erstellen

- Überprüfen Sie unten in der Basis Avatar-Komponente Ihres Avatars, ob die entsprechenden Build-Targets für Ihren Inhalt aktiviert sind. Basis unterstützt verschiedene Zielplattformen, darunter: Windows, Android, Mac, Linux und IOS.
- Drücken Sie die Schaltfläche 'Create Avatar Bee File' und warten Sie, bis dieser Vorgang abgeschlossen ist.

![Avatar Bee File erstellen](/img/avatars/7.png)

- Die Ergebnisse werden automatisch in einem neuen Fenster geöffnet, wie unten gezeigt.

![Build-Ergebnisse](/img/avatars/8.png)

- Je nach Hardware kann jede Plattform beim ersten Mal durchschnittlich ein bis zwei Minuten für den Build benötigen. Nachfolgende Builds sollten dank Caching weniger Zeit in Anspruch nehmen. Das Erstellen für alle Plattformen wird wahrscheinlich zusätzliche Zeit benötigen und die Gesamtdateigröße erhöhen.

<Callout type="info">
Gut zu wissen:

- Sie können bequem ein benutzerdefiniertes Passwort für Ihre generierte BEE-Datei verwenden. Das kann die Erinnerung erleichtern, wenn Sie ein einzelnes Passwort für verschiedene BEE-Dateien bevorzugen. 

- Wenn Sie möchten, können Sie auch den generierten .BEE-Dateinamen durch etwas leichter Merkbares ersetzen. Bedenken Sie, dass die Wiederverwendung desselben Namens dazu führt, dass jede vorherige Version durch die neue ersetzt wird!

![Avatar Bee File erstellen](/img/avatars/7a.png)
</Callout>

<Callout type="info">
Anekdote vom Januar 2026: Ein Avatar mit 18 Materialien, die Poiyomi URP mit 120.000 Polygonen verwenden, mit einer Paketgröße von 45 MB pro Plattform, benötigte auf einem Ryzen 7 9800x3D 9 Min. 43 Sek. für den Build für alle Zielplattformen.
</Callout>
