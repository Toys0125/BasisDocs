## Comment faire fonctionner les avatars utilisant des shaders Poiyomi dans BasisVR/URP

De nombreux avatars utilisent des shaders Poiyomi qui fonctionnaient à l'origine uniquement avec le Built-in Render Pipeline (BIRP) d'Unity. Cependant, maintenant, les shaders Poiyomi sont également pris en charge pour fonctionner avec le Universal Render Pipeline (URP) d'Unity, qui est le pipeline de rendu utilisé pour BasisVR.

La prise en charge des shaders Poiyomi pour URP est encore en bêta. Si vous avez un avatar qui a été conçu à l'origine avec des shaders Poiyomi BIRP, vous pouvez facilement les convertir pour être compatibles URP en suivant les étapes ci-dessous. Pour illustrer comment, cet exemple utilise [Hatsune Miku par Tomarudo](https://boothplorer.com/avatar/3226395).

- Téléchargez la dernière version depuis https://discord.gg/poiyomi #urp-release.

- Importez votre personnage dans la scène Unity. Si le shader semble différent de ce qui est attendu (par exemple, les couleurs sont manquantes ou il y a une couleur magenta en raison de shaders manquants ou similaires), vous devrez effectuer quelques petits ajustements pour corriger les choses.

![Fenêtre inspecteur montrant le bouton et les composants](/img/avatars/23.png)

- Sélectionnez chaque matériau dans le dossier de votre avatar et changez son shader pour utiliser Poiyomi Pro URP.

![Fenêtre inspecteur montrant le bouton et les composants](/img/avatars/24.png)

- Après avoir effectué les changements, votre avatar devrait s'afficher comme prévu.

![Fenêtre inspecteur montrant le bouton et les composants](/img/avatars/25.png)

<Callout type="info">
Les shaders Poiyomi pour URP sont compatibles avec les appareils PC et Android. 
</Callout>
