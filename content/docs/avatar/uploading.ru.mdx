## Загрузка вашего аватара

Basis позволяет загружать файл .BEE аватара по удалённому URL, что также необходимо для того, чтобы другие пользователи в сетевой сессии могли загружать ресурсы.

<Callout type="info">
Вы можете добавить пароль к аватару прямо в URL файла .BEE: `https://locationofBeefile.BEE#(Base64Encode(Password))`<br />
Вы добавляете `#` и строку пароля, закодированную в Base64.
</Callout>

Для этого вам нужна прямая ссылка на удалённое хранилище. В данном примере используется Google Drive.

(Google Drive поддерживает прямые ссылки на файлы до 100 МБ благодаря политике проверки на вирусы.)

Загрузите **файл BEE** на Google Drive или любой хостинг, позволяющий прямые ссылки на файлы без входа в систему.

- Создайте папку на Google Drive и установите настройки общего доступа «Все, у кого есть ссылка».

Теперь при перетаскивании файлов в эту папку не нужно настраивать разрешения для каждого файла отдельно.

- Нажмите на гамбургер-меню в верхнем правом углу только что загруженного ресурса.
- Найдите «Поделиться» и нажмите «Копировать ссылку».

Нам нужно изучить URL, чтобы получить необходимые данные для файла BEE.

Пример:
```
https://drive.google.com/file/d/YourIDBringsAllTheLagToMyYard/view?usp=sharing
```

Нам нужно скопировать уникальный идентификатор из URL и добавить его к следующему:
```
https://drive.google.com/uc?export=download&id=
```

Пример:
```
https://drive.google.com/uc?export=download&id=YourIDBringsAllTheLagToMyYard
```

## Загрузка в Basis
<Steps>

<Step>
Запустите Basis, откройте меню и выберите Библиотека.

![Меню Basis](/img/avatars/11.png)
</Step>

<Step>
В меню Библиотека выберите «Добавить новый контент» в верхнем правом углу окна.

![Кнопка добавления аватара](/img/avatars/12.png)
</Step>

<Step>
Вставьте **URL файла BEE** в поле URL файла BEE и **СУПЕРСЕКРЕТНЫЙ ЗАШИФРОВАННЫЙ ПАРОЛЬ** в поле пароля файла BEE, затем нажмите «Добавить» ниже.

![URL и пароль файла BEE](/img/avatars/13.png)
</Step>

<Step>
Содержимое файла BEE будет проверено перед добавлением в вашу библиотеку.

![Кнопка добавления файла BEE](/img/avatars/14.png)
</Step>
</Steps>

## Управление аватарами

Вы можете найти недавно добавленный аватар в Библиотеке.

![Новый аватар в библиотеке](/img/avatars/15.png)

Аватары можно Загрузить, Удалить или Поделиться с другими в мире, нажав соответствующие кнопки в нижней части окна.

![Детали аватара](/img/avatars/16.png)

Кроме того, здесь вы также увидите несколько полезных информационных панелей:

- **(1) Информация об аватаре** — панель деталей отображает временную метку компиляции, Название и Описание
- **(2) Совместимость платформ** — значки указывают, какие ОС-платформы поддерживают этот аватар
- **(3) Статистика** — отображает количество Треугольников, Материалов и Костей
