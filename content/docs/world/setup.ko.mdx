## Unity Editor 설정

- 최신 지원 Unity 에디터 버전 설치 및 Basis 프레임워크 획득에 대한 [시작하기](/ko/docs#installing-unity) 섹션을 참조하세요.
- Unity Hub를 열고 기존 프로젝트를 추가한 후, 압축 해제/클론된 Basis 프로젝트 폴더를 선택합니다.
- Unity 프로젝트가 로드되면 월드 파일을 로드할 수 있습니다:
- 다음 중 하나를 선택하세요:
  - 프로젝트 계층 구조에서 `Assets`를 찾고, `Assets` 안의 빈 공간을 우클릭한 후 `Show in Explorer`를 선택합니다.
  - 월드 파일 [메시/머티리얼/텍스처 등]을 복사해 넣습니다.
- 또는
  - Assets 메뉴에서 `Import Package > Custom Package`를 통해 파일이 포함된 .unity 패키지를 가져옵니다.
- Unity 창으로 돌아와 가져오기 프로세스가 완료될 때까지 기다립니다.

<Callout type="info">
콘텐츠를 "Assets > _UserContent" 새 디렉토리에 배치하면 프로젝트의 기본 .gitignore 파일이 git에서 이 폴더의 변경사항을 숨깁니다.
</Callout>
