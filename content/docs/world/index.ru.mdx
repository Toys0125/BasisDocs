## Обзор

Миры — это окружения, в которых происходят социальные VR-переживания в Basis. Данная документация проведёт вас через настройку, создание и тестирование собственных пользовательских миров с помощью Unity и Basis SDK.

Basis использует формат файла `.BEE` для миров, который упаковывает скомпилированную сцену и метаданные в один файл. Миры поддерживают пользовательские меши, материалы, освещение и точки появления, а также могут размещаться удалённо для мультиплеерных опытов.

## Ключевые функции

- **Пользовательские окружения**: Импортируйте любые 3D-меши, материалы и текстуры
- **Точки появления**: Определите, где игроки входят в ваш мир
- **Мультиплатформенность**: Сборка для Windows, Linux и Android
- **Готово к сети**: Спроектировано для мультиплеерных опытов
- **Формат BEE**: Простая упаковка для лёгкого распространения

## Требования

- **Unity**: Unity 6 (последняя поддерживаемая версия)
- **Ассеты**: 3D-меши с коллайдерами для навигации
- **Компоненты**: Компонент Basis Scene для конфигурации сцены
- **Хостинг**: Удалённый URL-хостинг для мультиплеера (необязательно для локального тестирования)

## Обзор рабочего процесса

1. **Импортируйте** ассеты мира в Unity
2. **Настройте** компонент Basis Scene и точки появления
3. **Создайте** файл .BEE для целевых платформ
4. **Тестируйте** локально с помощью префаба BasisFramework
5. **Загрузите** на хостинг-сервис для мультиплеера (необязательно)
6. **Загрузите** в Basis, используя URL и пароль
