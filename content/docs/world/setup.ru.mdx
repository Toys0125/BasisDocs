## Настройка редактора Unity

- Согласно разделу [Начало работы](/ru/docs#installing-unity) для установки последней поддерживаемой версии редактора Unity и получения фреймворка Basis.
- Откройте Unity Hub и добавьте существующий проект, выберите папку извлечённого/клонированного проекта Basis.
- После загрузки проекта Unity вы можете загрузить файлы вашего мира:
- Либо,
  - Найдите `Assets` в иерархии проекта, внутри `Assets` щёлкните правой кнопкой мыши на пустом месте и выберите `Show in Explorer`.
  - Скопируйте файлы вашего мира [Меши/Материалы/Текстуры и т.д.].
- или
  - Импортируйте пакет .unity с вашими файлами через меню Assets: `Import Package > Custom Package`.
- Вернитесь в окно Unity и дождитесь завершения процесса импорта.

<Callout type="info">
Если вы размещаете контент в новом каталоге "Assets > _UserContent", файл .gitignore по умолчанию скроет изменения в этой папке от git.
</Callout>
