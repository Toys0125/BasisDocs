## 씬 버튼을 사용하여 소품 스폰하기

Runtime Loader 컴포넌트는 런타임 시 *Basis\Basis Server\BasisServerConsole\bin\Debug\net9.0\initialresources*에 있는 모든 소품을 생성합니다. 씬의 버튼을 사용하여 소품을 생성하는 것도 가능합니다. 방법은 다음과 같습니다:

- 씬에 캔버스 오브젝트를 추가합니다. 이전 섹션에서 보여준 것과 유사한 관련 컴포넌트를 첨부해야 합니다. 소품의 .bee 파일을 생성하고 원격 서버에 업로드하기 위해 Basis Prop 컴포넌트를 추가하는 것을 잊지 마세요.

![캔버스와 컴포넌트가 있는 Inspector 창](/img/props/6A.png)

- TMP 버튼을 만듭니다.

![버튼과 컴포넌트가 있는 Inspector 창](/img/props/6B.png)

- Basis Prop 컴포넌트를 추가하고, Asset Bundle Name을 입력한 후 'Create Prop Bee File' 버튼을 누릅니다. 생성된 .bee 파일은 Basis Runtime Loader가 씬에 소품을 로드하는 데 사용됩니다. 이 예시에서는 이전 섹션의 네트워크 동기화된 큐브를 다시 사용합니다.

![동기화된 큐브와 컴포넌트가 있는 Inspector 창](/img/props/6B-.png)

- 빈 Gameobject를 만들고 Runtime Loader 컴포넌트를 첨부합니다. 이것은 소품을 스폰하는 데 사용됩니다. 씬 시작 시 소품이 생성되는 것을 방지하려면 inspector에서 컴포넌트를 비활성화하세요.

![비활성화된 Runtime Loader가 있는 Inspector 창](/img/props/6C.png)

<Callout type="info">
소품을 네트워크 동기화하려면 네트워킹 동기화 컴포넌트를 추가하세요. 예를 들어, 이전에 보여준 대로 Basis Object Network Syncing 컴포넌트를 사용하여 소품 이동을 동기화할 수 있습니다.

자신만의 네트워크 동기화 코드를 만들 수도 있습니다. 영감을 얻고 시작하는 데 도움이 되도록, 프로젝트 내에서 찾을 수 있는 예시 패키지 *Basis Ported Pooltable*과 *Basis Vehicles*를 참조할 수 있습니다.
</Callout>

- 버튼을 선택하고 onClick() 이벤트를 추가합니다. Runtime Loader Gameobject를 할당하고, 버튼이 클릭될 때 호출할 메서드로 BasisRuntimeLoader.LoadNow를 선택합니다.

![RuntimeLoader를 호출하는 버튼 이벤트가 있는 Inspector 창](/img/props/6D.png)

- Load Config 에디터를 엽니다.

- 생성된 소품 비밀번호와 .bee url을 입력합니다.

- 늦게 참가하는 플레이어에게도 동기화된 변경사항이 유지되도록 Persistent를 선택합니다.

- Save XML을 클릭합니다.

![로드 구성이 있는 Inspector 창](/img/props/6E.png)

- 아직 하지 않았다면 빈 Gameobject를 만들고 Basis Scene 컴포넌트를 첨부하여 관련 필드를 채웁니다. 설정 후 'Create Scene Bee File'을 클릭합니다.

![Basis 씬이 있는 Inspector 창](/img/props/6F.png)

- Load Config 에디터를 다시 엽니다.

- 생성된 씬 비밀번호와 .bee url을 입력합니다.

- 늦게 참가하는 플레이어에게도 동기화된 변경사항이 유지되도록 Persistent를 선택합니다.

- Save XML을 클릭합니다.

![버튼과 컴포넌트가 있는 Inspector 창](/img/props/6G.png)

- 'Initialization' 씬을 열고 play를 누릅니다. 이것은 생성된 씬과 네트워크 오브젝트를 초기화합니다. 다음에 basis 애플리케이션을 실행하면서 서버를 통해 씬을 로드하면 버튼을 통해 소품을 스폰할 수 있어야 합니다.

![버튼과 컴포넌트가 있는 Inspector 창](/img/props/6H.png)

<Callout type="info">
참고: 로컬 및 원격 플레이어 모두 큐브를 스폰할 수 있습니다!
</Callout>
