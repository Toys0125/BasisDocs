## 씬 테스트

<Callout type="warn">
프로젝트 파일에서 `BasisFramework` .prefab 에셋을 선택하세요. **이 프리팹을 현재 씬에 인스턴스화하지 마세요! Basis SDK가 자동으로 처리합니다!**
</Callout>

- AssetBundle이 생성되면 `Packages > Basis SDK > Prefabs > Boot`에서 찾을 수 있는 `BasisFramework` 프리팹을 통해 로드할 수 있습니다.

![BasisFramework 프리팹 위치](/img/worlds/7.png)

- 새 AssetBundle의 위치를 `Remote BEE File Location` 필드에 입력하고, `/AssetBundles/` 디렉토리에 생성된 `dontuploadmepassword.txt` 파일의 `Unlock Password`와 함께 `Use Scene Provided Here`를 활성화합니다.

![BasisFramework 구성](/img/worlds/8.png)

<Callout type="info">
참고: *Use Scene Provided Here*가 활성화되면 *Remote Bee File Location*에 지정된 씬을 로드합니다. 비활성화되면 Basis Config Loader를 통해 생성된 XML 씬을 로드합니다 (있는 경우).
</Callout>

- `Packages > Basis SDK > Scenes`에서 `Initialization` 씬을 로드한 후 Play를 누릅니다.

![초기화 씬](/img/worlds/9.png)

- 사용자 이름을 입력하고 "Connect"를 누릅니다. 로컬 머신에서 [서버](/ko/docs/server)가 실행 중인 경우 localhost를 사용할 수 있습니다.

- 그러면 월드의 스폰 위치 또는 연결된 서버에 로드됩니다.
