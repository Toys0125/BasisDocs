## 主要用語

**ローカルプレイヤー**  
_あなた_（ユーザーまたはクライアント）を表すローカルアバターデータ。

**リモートプレイヤー**  
サーバーからデータが送られてくる他のプレイヤーを表します。

**Basisクライアント**  
サーバーと通信するために作成するアプリケーション。

**モーションデータ**  
プレイヤーの骨の位置、ワールド内の位置と回転を表すデータ。
LASMデータ構造のネットワーク直列化可能なフォーマットで保存されます。

**LASM**  
Local Avatar Sync Messageの略で、ネットワーク経由で送信されるプレイヤーの筋肉、位置、回転の圧縮データを含むデータ構造。
