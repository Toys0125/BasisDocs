## Belangrijkste termen

**Lokale speler**  
De lokale avatardata die _jou_ vertegenwoordigt (de gebruiker of client).

**Externe speler**  
Vertegenwoordigt andere spelers wiens data door de server naar jou wordt verzonden.

**Basis-client**  
De applicatie die je schrijft en die met de server communiceert.

**Bewegingsdata**  
De data die de botposities van de speler vertegenwoordigt, evenals de positie en rotatie in de wereld.
Opgeslagen in een netwerk-serialiseerbaar formaat in de LASM-datastructuur.

**LASM**  
Staat voor Local Avatar Sync Message, de datastructuur die gecomprimeerde data bevat
voor de spierposities, positie en rotatie van een speler die via het netwerk worden verzonden.
