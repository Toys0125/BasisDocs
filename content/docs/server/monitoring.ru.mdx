## Конечная точка проверки состояния

При локальном запуске сервера вы можете проверить его состояние, посетив конечную точку проверки состояния по адресу:
http://localhost:10666/health

```json title="http://localhost:10666/health"
{
  "listening": true,
  "visitors": "0",
  "capacity": "1024",
  "sent": "0",
  "recv": "0",
  "currentTime": "2025-05-09T20:19:03.200Z",
  "startTime": "2025-05-09T20:19:00.049Z",
  "version": "6"
}
```

| Переменная    | Определение                         |
|---------------|--------------------------------------|
| `listening`   | Принимает ли сервер подключения |
| `visitors`    | Количество игроков, подключённых в данный момент |
| `capacity`    | Максимальное количество игроков, допустимых на сервере |
| `sent`        | Общий объём отправленных данных (в байтах?) |
| `recv`        | Общий объём полученных данных (в байтах?) |
| `currentTime` | Текущее время сервера в формате ISO 8601 |
| `startTime`   | Время запуска сервера в формате ISO 8601 |
| `version`     | Номер версии сервера. Важно: код клиента должен совпадать с ожидаемой ServerVersion |
