## 상태 확인 엔드포인트

서버를 로컬에서 실행할 때 다음 상태 확인 엔드포인트를 방문하여 서버 상태를 확인할 수 있습니다:
http://localhost:10666/health

```json title="http://localhost:10666/health"
{
  "listening": true,
  "visitors": "0",
  "capacity": "1024",
  "sent": "0",
  "recv": "0",
  "currentTime": "2025-05-09T20:19:03.200Z",
  "startTime": "2025-05-09T20:19:00.049Z",
  "version": "6"
}
```

| 변수          | 정의                                |
|---------------|--------------------------------------|
| `listening`   | 서버가 연결을 수신 중인지 여부 |
| `visitors`    | 현재 플레이어(방문자) 수 |
| `capacity`    | 서버에서 허용되는 최대 플레이어 수 |
| `sent`        | 전송된 총 데이터 양 (바이트?) |
| `recv`        | 수신된 총 데이터 양 (바이트?) |
| `currentTime` | ISO 8601 형식의 현재 서버 시간 |
| `startTime`   | ISO 8601 형식의 서버 시작 시간 |
| `version`     | 서버의 버전 번호. 중요: 클라이언트 코드가 예상 ServerVersion과 일치해야 합니다 |
