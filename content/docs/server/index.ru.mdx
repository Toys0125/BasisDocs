## Обзор

Сервер Basis является сетевой основой для вашего социального VR-опыта. Данная документация проведёт вас через настройку, конфигурирование и управление собственным экземпляром сервера Basis.

Сервер построен на .NET 9.0 и использует LiteNetLib для низкоуровневого сетевого взаимодействия. Он спроектирован для обработки нескольких одновременных игроков с эффективным использованием полосы пропускания и поддерживает как развёртывание через Docker, так и нативную компиляцию на Windows и Linux.

## Ключевые функции

- **Многопоточная производительность**: Хорошо работает даже на двух ядрах, масштабируется с увеличением числа ядер
- **Гибкое развёртывание**: Поддержка Docker, Windows и Linux
- **Аутентификация**: Верификация личности на основе DID
- **Оптимизация полосы пропускания**: Серверные и клиентские системы редукции
- **Мониторинг здоровья**: Встроенная конечная точка HTTP для проверки состояния
- **Загружаемые ресурсы**: Динамическая загрузка миров и объектов

## Требования

- **CPU**: Минимум 2 ядра (больше ядер улучшают производительность)
- **Сеть**: Рекомендуется высокая пропускная способность для социального VR
- **Порты**:
  - 4296/udp (протокол Basis)
  - 10666/tcp (проверки состояния)
  - 1234/tcp (метрики Prometheus)

## Версия сервера

Данная документация охватывает **Basis Server Version 6** из ветки LTS (май 2025 года).

<Callout type="warn">
Код клиента и сервера **должен совпадать** по номеру версии. `ServerVersion` можно найти в `BasisNetworkCore\BasisNetworkVersion.cs`.
</Callout>
