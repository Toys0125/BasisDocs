## Panoramica

Il Basis Server è la spina dorsale di rete per la tua esperienza di VR sociale. Questa documentazione ti guiderà nella configurazione e nella gestione della tua istanza del server Basis.

Il server è costruito su .NET 9.0 e utilizza LiteNetLib per il networking di basso livello. È progettato per gestire più giocatori contemporaneamente con un utilizzo efficiente della larghezza di banda e supporta sia le implementazioni Docker che la compilazione nativa su Windows e Linux.

## Funzionalità principali

- **Prestazioni multi-thread**: Funziona bene anche con soli due core, scala con più core
- **Implementazione flessibile**: Supporto per Docker, Windows e Linux
- **Autenticazione**: Verifica dell'identità basata su DID
- **Ottimizzazione della larghezza di banda**: Sistemi di riduzione lato server e client
- **Monitoraggio della salute**: Endpoint HTTP di controllo della salute integrato
- **Risorse caricabili**: Caricamento dinamico di mondi e oggetti

## Requisiti

- **CPU**: Minimo 2 core (più core migliorano le prestazioni)
- **Rete**: Larghezza di banda elevata raccomandata per la VR sociale
- **Porte**:
  - 4296/udp (protocollo Basis)
  - 10666/tcp (Controlli della salute)
  - 1234/tcp (Metriche Prometheus)

## Versione del server
