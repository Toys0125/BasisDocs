## ヘルスチェックエンドポイント

サーバーをローカルで実行している場合、ヘルスチェックエンドポイントにアクセスしてサーバーの状態を確認できます：
http://localhost:10666/health

```json title="http://localhost:10666/health"
{
  "listening": true,
  "visitors": "0",
  "capacity": "1024",
  "sent": "0",
  "recv": "0",
  "currentTime": "2025-05-09T20:19:03.200Z",
  "startTime": "2025-05-09T20:19:00.049Z",
  "version": "6"
}
```

| 変数          | 定義                                |
|---------------|--------------------------------------|
| `listening`   | サーバーが接続を待ち受けているか |
| `visitors`    | 現在接続しているプレイヤーの数 |
| `capacity`    | サーバーで許可される最大プレイヤー数 |
| `sent`        | 送信データの総量（バイト単位？）|
| `recv`        | 受信データの総量（バイト単位？）|
| `currentTime` | ISO 8601形式のサーバーの現在時刻 |
| `startTime`   | ISO 8601形式のサーバーの起動時刻 |
| `version`     | サーバーのバージョン番号。重要：クライアントコードは期待されるServerVersionと一致する必要があります |
