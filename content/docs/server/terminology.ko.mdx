## 주요 용어

**로컬 플레이어**  
_당신_(사용자 또는 클라이언트)을 나타내는 로컬 아바타 데이터.

**원격 플레이어**  
서버가 당신에게 데이터를 전송하는 다른 플레이어를 나타냅니다.

**Basis 클라이언트**  
서버와 통신할 작성 중인 애플리케이션.

**모션 데이터**  
플레이어 본 위치와 월드에서의 위치 및 회전을 나타내는 데이터.
LASM 데이터 구조의 네트워크 직렬화 가능 형식으로 저장됩니다.

**LASM**  
Local Avatar Sync Message의 약자로, 네트워크를 통해 전송되는 플레이어의 근육, 위치 및 회전에 대한 압축된 데이터를 포함하는 데이터 구조입니다.
