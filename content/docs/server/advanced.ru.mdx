## Система редукции сервера

По мере роста числа игроков на сервере требования к пропускной способности быстро растут. Система SRS пытается снизить сетевую нагрузку и улучшить производительность, замедляя частоту отправки данных о движении игрокам в зависимости от расстояния.

Уравнение, определяющее, насколько дольше сервер отправляет данные о движении пользователям в зависимости от расстояния:

```csharp
int adjustedInterval = (int)(Configuration.BSRSMillisecondDefaultInterval * (Configuration.BSRBaseMultiplier + (activeDistance * Configuration.BSRSIncreaseRate)));
```
