Bienvenue sur le wiki Basis !

Cette page vous guidera à travers les étapes minimales requises pour démarrer un projet Basis, y compris des guides pratiques pour créer des bundles d'assets Avatar et World.

## Configuration système requise

https://docs.unity3d.com/Manual/system-requirements.html

## Installer Unity

- Hub - https://unity.com/download
- Standalone - https://unity.com/releases/editor/archive

### Versions Unity compatibles

Selon la branche utilisée, veuillez respecter la version de l'éditeur Unity sur laquelle le framework est basé :

✓ [Developer](https://github.com/BasisVR/Basis/blob/developer/Basis/ProjectSettings/ProjectVersion.txt#L1)  
✓ [Long-Term-Support](https://github.com/BasisVR/Basis/blob/long-term-support/Basis/ProjectSettings/ProjectVersion.txt#L1)

### Modules Unity supplémentaires

Ajoutez les modules suivants via l'écran "Ajouter des modules" lors de l'installation de la version de l'éditeur Unity :

```ts tab="Windows"
✓ Android Build Support  
   ↳ ✓ OpenJDK  
   ↳ ✓ Android SDK & NDK Tools  
✓ IOS Build Support  
✓ Linux Build Support (Mono) 
✓ Mac Build Support (Mono)
✓ Windows Build Support (IL2CPP)
```

```ts tab="Mac"
✓ Android Build Support  
   ↳ ✓ OpenJDK  
   ↳ ✓ Android SDK & NDK Tools  
✓ IOS Build Support  
✓ Linux Build Support (Mono)
✓ Mac Build Support (IL2CPP)
✓ Windows Build Support (Mono)
```

```ts tab="Linux"
✓ Android Build Support  
   ↳ ✓ OpenJDK  
   ↳ ✓ Android SDK & NDK Tools  
✓ IOS Build Support  
✓ Linux Build Support (IL2CPP)
✓ Mac Build Support (Mono)
✓ Windows Build Support (Mono)
```

<Callout type="info">
Même si vous ne créez que du contenu généré par l'utilisateur pour les avatars et les mondes, vous aurez quand même besoin de ces modules supplémentaires pour rendre ce contenu disponible sur ces plateformes.
</Callout>

<Callout type="info">
Si vous avez déjà la bonne version de Unity installée mais que l'option "Ajouter des modules" n'est pas disponible, elle a probablement été installée en dehors de Unity Hub. Veuillez réinstaller la bonne version de Unity via Unity Hub.
</Callout>

## Obtenir le framework Basis

### Git

Vous pouvez directement [cloner](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository) les branches `developer` ou `long-term-support` disponibles sur https://github.com/BasisVR/Basis, ou [forker](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/working-with-forks/fork-a-repo) dans votre propre dépôt local, puis ouvrir le projet Unity résultant sur votre ordinateur.

### Téléchargement ZIP direct

Pour ceux qui souhaitent éviter Git :

- Sélectionnez le menu déroulant "Code" en haut à droite, puis Télécharger ZIP.
- Décompressez le ZIP dans n'importe quel dossier de votre choix.
- Ajoutez le sous-dossier Basis comme projet existant dans Unity Hub.
