¡Bienvenido al wiki de Basis!

Esta página te ayudará a proporcionar los pasos mínimos necesarios para poner en marcha un proyecto Basis, incluyendo guías para crear bundles de assets de Avatar y World.

## Requisitos del sistema actuales

https://docs.unity3d.com/Manual/system-requirements.html

## Instalar Unity

- Hub - https://unity.com/download
- Standalone - https://unity.com/releases/editor/archive

### Versiones de Unity compatibles

Según la rama utilizada, por favor observa la versión del editor Unity sobre la que se construye el framework:

✓ [Developer](https://github.com/BasisVR/Basis/blob/developer/Basis/ProjectSettings/ProjectVersion.txt#L1)  
✓ [Long-Term-Support](https://github.com/BasisVR/Basis/blob/long-term-support/Basis/ProjectSettings/ProjectVersion.txt#L1)

### Módulos adicionales de Unity

Añade los siguientes módulos a través de la pantalla "Añadir módulos" al instalar la versión del editor Unity:

```ts tab="Windows"
✓ Android Build Support  
   ↳ ✓ OpenJDK  
   ↳ ✓ Android SDK & NDK Tools  
✓ IOS Build Support  
✓ Linux Build Support (Mono) 
✓ Mac Build Support (Mono)
✓ Windows Build Support (IL2CPP)
```

```ts tab="Mac"
✓ Android Build Support  
   ↳ ✓ OpenJDK  
   ↳ ✓ Android SDK & NDK Tools  
✓ IOS Build Support  
✓ Linux Build Support (Mono)
✓ Mac Build Support (IL2CPP)
✓ Windows Build Support (Mono)
```

```ts tab="Linux"
✓ Android Build Support  
   ↳ ✓ OpenJDK  
   ↳ ✓ Android SDK & NDK Tools  
✓ IOS Build Support  
✓ Linux Build Support (IL2CPP)
✓ Mac Build Support (Mono)
✓ Windows Build Support (Mono)
```

<Callout type="info">
Incluso si solo estás creando contenido generado por el usuario para la creación de Avatars y Worlds, seguirás necesitando estos módulos adicionales para que el contenido esté disponible en estas plataformas.
</Callout>

<Callout type="info">
Si ya tienes instalada la versión correcta de Unity pero la opción "Añadir módulos" no está disponible, probablemente fue instalada externamente desde Unity Hub. Por favor, reinstala la versión correcta de Unity desde Unity Hub.
</Callout>

## Obtener el framework Basis

### Git

Puedes directamente [clonar](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository) las ramas `developer` o `long-term-support` en https://github.com/BasisVR/Basis o [hacer un fork](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/working-with-forks/fork-a-repo) en tu propio repositorio local, y luego abrir el proyecto Unity resultante en tu computadora.

### Descarga directa de ZIP

Para quienes deseen prescindir de Git:

- Selecciona el menú desplegable "Code" en la parte superior derecha y luego Descargar ZIP.
- Descomprime el ZIP en cualquier carpeta que desees.
- Añade la subcarpeta Basis como proyecto existente en Unity Hub.
