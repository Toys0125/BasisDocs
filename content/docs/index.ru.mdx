Эта страница поможет вам выполнить минимальные шаги для запуска проекта Basis, включая руководства по созданию бандлов ассетов Avatar и World.

https://docs.unity3d.com/Manual/system-requirements.html

## Установка Unity

- Hub - https://unity.com/download
- Standalone - https://unity.com/releases/editor/archive

### Совместимые версии Unity

В зависимости от используемой ветки, пожалуйста, используйте версию редактора Unity, на которой основан фреймворк:

✓ [Developer](https://github.com/BasisVR/Basis/blob/developer/Basis/ProjectSettings/ProjectVersion.txt#L1)  
✓ [Long-Term-Support](https://github.com/BasisVR/Basis/blob/long-term-support/Basis/ProjectSettings/ProjectVersion.txt#L1)

### Дополнительные модули Unity

Добавьте следующие модули через экран «Добавить модули» при установке версии редактора Unity:

```ts tab="Windows"
✓ Android Build Support  
   ↳ ✓ OpenJDK  
   ↳ ✓ Android SDK & NDK Tools  
✓ IOS Build Support  
✓ Linux Build Support (Mono) 
✓ Mac Build Support (Mono)
✓ Windows Build Support (IL2CPP)
```

```ts tab="Mac"
✓ Android Build Support  
   ↳ ✓ OpenJDK  
   ↳ ✓ Android SDK & NDK Tools  
✓ IOS Build Support  
✓ Linux Build Support (Mono)
✓ Mac Build Support (IL2CPP)
✓ Windows Build Support (Mono)
```

```ts tab="Linux"
✓ Android Build Support  
   ↳ ✓ OpenJDK  
   ↳ ✓ Android SDK & NDK Tools  
✓ IOS Build Support  
✓ Linux Build Support (IL2CPP)
✓ Mac Build Support (Mono)
✓ Windows Build Support (Mono)
```

<Callout type="info">
Даже если вы создаёте только пользовательский контент для аватаров и миров, вам всё равно потребуются эти дополнительные модули для обеспечения доступности контента на этих платформах.
</Callout>

<Callout type="info">
Если у вас уже установлена правильная версия Unity, но опция «Добавить модули» недоступна, вероятно, она была установлена вне Unity Hub. Пожалуйста, переустановите правильную версию Unity через Unity Hub.
</Callout>

## Получение фреймворка Basis

### Git

Вы можете напрямую [клонировать](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository) ветки `developer` или `long-term-support` с https://github.com/BasisVR/Basis или [форкнуть](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/working-with-forks/fork-a-repo) в собственный локальный репозиторий, а затем открыть полученный проект Unity на своём компьютере.

### Прямое скачивание ZIP

Для тех, кто хочет обойтись без Git:

- Выберите выпадающее меню «Code» в правом верхнем углу, затем «Скачать ZIP».
- Распакуйте ZIP в любую папку.
- Добавьте подпапку Basis как существующий проект в Unity Hub.
