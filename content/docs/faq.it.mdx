## Domande frequenti

### Che cos'è BasisVR?

BasisVR è un framework software che fornisce agli sviluppatori integrazioni tecnologiche avanzate e approcci di alto livello per creare esperienze sociali in Realtà Virtuale.

<Callout type="info">
Nota: ***Basis non è una piattaforma Social VR***. Include primitive utili, ma il progetto principale non è pensato per creare il tipo di infrastruttura sociale scalabile che le piattaforme Social VR richiedono. Non ostacola i creatori che desiderano costruire la propria, ma il progetto stesso distribuisce framework e demo.
</Callout>

### Quali sono i requisiti di sistema?

BasisVR richiede:
- Un PC con supporto VR o un visore VR standalone
- Unity per lo sviluppo
- Controller VR compatibili

### Come posso iniziare?

1. Installa BasisVR
2. Configura il tuo avatar
3. Crea o unisciti a un mondo
4. Inizia a esplorare!

Consulta la [Guida introduttiva](/docs) per istruzioni dettagliate sulla configurazione.

### Posso creare i miei avatar?

Sì! BasisVR supporta avatar personalizzati. Consulta la [Guida agli Avatar](/it/docs/avatar) per ulteriori informazioni sulla creazione e l'importazione di avatar personalizzati.

### Come posso creare il mio mondo?

Puoi creare mondi personalizzati usando Unity. Consulta la [Guida alla Creazione di Mondi](/it/docs/world) per istruzioni passo-passo.

### Come posso contribuire?

Accogliamo contributi! Consulta la [Guida ai Contributi](/it/docs/contributing) per ulteriori informazioni su come partecipare.

### Qual è la differenza tra contributi ai componenti principali e contributi aggiuntivi?

I componenti principali includono i sistemi che costituiscono le funzionalità chiave di Basis. I contributi (o moduli) includono funzionalità extra che, pur essendo utili, non sono richieste per impostazione predefinita. Pertanto, i contributi possono fornire funzionalità al di fuori dell'ambito dell'offerta principale di Basis, che possono risultare preziose per gli sviluppatori che usano Basis.

### La condivisione di file BEE compromette le credenziali del server di hosting dei contenuti remoti?

No! Puoi condividere i tuoi file BEE con gli amici, ad esempio. Un file BEE comprende l'asset Basis (anche per più piattaforme), esclusa la chiave di crittografia usata per decifrarlo. In altre parole, è essenzialmente un multi-asset-bundle. Se condividi la password di un file BEE, chiunque la possieda potrà accedere al contenuto all'interno del file BEE (ovvero i specifici UGC come prop, avatar, mondi).

## Come posso connettermi a un server locale da dispositivi diversi?

Esistono diverse opzioni per connettersi a un server locale da dispositivi diversi. Ad esempio, puoi abilitare il port forwarding sul tuo router. In alternativa, se preferisci non modificare il tuo router, puoi installare [`Tailscale`](https://tailscale.com/download), che è molto semplice da configurare.

## Dove posso configurare un server remoto?

Sono disponibili diversi servizi cloud per ospitare i server. Alcuni esempi includono: Bunny, DigitalOcean, CloudFlare, AWS, Microsoft Azure, Google Cloud e altri.
