欢迎来到 Basis 维基！

本页将帮助您了解启动并运行 Basis 项目所需的最小步骤，包括创建 Avatar 和 World 资产包的操作指南。

## 当前系统要求

https://docs.unity3d.com/Manual/system-requirements.html

## 安装 Unity

- Hub - https://unity.com/download
- 独立版 - https://unity.com/releases/editor/archive

### 兼容的 Unity 版本

根据所使用的分支，请注意框架所基于的 Unity 编辑器版本：

✓ [Developer](https://github.com/BasisVR/Basis/blob/developer/Basis/ProjectSettings/ProjectVersion.txt#L1)  
✓ [Long-Term-Support](https://github.com/BasisVR/Basis/blob/long-term-support/Basis/ProjectSettings/ProjectVersion.txt#L1)

### 附加 Unity 模块

在安装 Unity 编辑器版本时，通过"添加模块"界面添加以下模块：

```ts tab="Windows"
✓ Android Build Support  
   ↳ ✓ OpenJDK  
   ↳ ✓ Android SDK & NDK Tools  
✓ IOS Build Support  
✓ Linux Build Support (Mono) 
✓ Mac Build Support (Mono)
✓ Windows Build Support (IL2CPP)
```

```ts tab="Mac"
✓ Android Build Support  
   ↳ ✓ OpenJDK  
   ↳ ✓ Android SDK & NDK Tools  
✓ IOS Build Support  
✓ Linux Build Support (Mono)
✓ Mac Build Support (IL2CPP)
✓ Windows Build Support (Mono)
```

```ts tab="Linux"
✓ Android Build Support  
   ↳ ✓ OpenJDK  
   ↳ ✓ Android SDK & NDK Tools  
✓ IOS Build Support  
✓ Linux Build Support (IL2CPP)
✓ Mac Build Support (Mono)
✓ Windows Build Support (Mono)
```

<Callout type="info">
即使您只是为 Avatar 和 World 创建用户生成内容，也仍然需要这些额外模块，以便在这些平台上提供内容。
</Callout>

<Callout type="info">
如果您已安装了正确的 Unity 版本，但"添加模块"选项不可用，则可能是从 Unity Hub 外部安装的。请通过 Unity Hub 重新安装正确的 Unity 版本。
</Callout>

## 获取 Basis 框架

### Git

您可以直接[克隆](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository) https://github.com/BasisVR/Basis 上的 `developer` 或 `long-term-support` 分支，或将其[复刻](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/working-with-forks/fork-a-repo)到您自己的本地仓库，然后在本地计算机上打开生成的 Unity 项目。

### 直接下载 ZIP

对于希望跳过 Git 体验的用户：

- 选择右上角的"Code"下拉菜单，然后点击"下载 ZIP"。
- 将 ZIP 解压到任意文件夹。
- 将 Basis 子文件夹作为现有项目添加到 Unity Hub。
