Welkom op de Basis-wiki!

Deze pagina biedt de minimaal vereiste stappen om een Basis-project op te zetten en te laten draaien, inclusief handleidingen voor het maken van Avatar- en Wereld-assetbundels.

## Huidige systeemvereisten

https://docs.unity3d.com/Manual/system-requirements.html

## Unity installeren

- Hub - https://unity.com/download
- Standalone - https://unity.com/releases/editor/archive

### Compatibele Unity-versies

Afhankelijk van de gebruikte branch, let op de versie van de Unity Editor waarop het framework is gebouwd:

✓ [Developer](https://github.com/BasisVR/Basis/blob/developer/Basis/ProjectSettings/ProjectVersion.txt#L1)  
✓ [Long-Term-Support](https://github.com/BasisVR/Basis/blob/long-term-support/Basis/ProjectSettings/ProjectVersion.txt#L1)

### Aanvullende Unity-modules

Voeg de volgende modules toe via het scherm "Modules toevoegen" bij het installeren van de Unity Editor-versie:

```ts tab="Windows"
✓ Android Build Support  
   ↳ ✓ OpenJDK  
   ↳ ✓ Android SDK & NDK Tools  
✓ IOS Build Support  
✓ Linux Build Support (Mono) 
✓ Mac Build Support (Mono)
✓ Windows Build Support (IL2CPP)
```



<Callout type="info">
Zelfs als je alleen gebruikersgegenereerde content maakt voor Avatar en Wereld, heb je deze aanvullende modules nog steeds nodig om deze content beschikbaar te maken op deze platformen.
</Callout>

<Callout type="info">
Als je de juiste Unity-versie al hebt geïnstalleerd maar de optie "Modules toevoegen" niet beschikbaar is, is deze waarschijnlijk buiten Unity Hub geïnstalleerd. Installeer de juiste Unity-versie opnieuw via Unity Hub.
</Callout>

## Het Basis-framework verkrijgen

### Git

Je kunt de `developer`- of `long-term-support`-branches op https://github.com/BasisVR/Basis rechtstreeks [klonen](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository) of [forken](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/working-with-forks/fork-a-repo) naar je eigen lokale repository, en vervolgens het resulterende Unity-project op je computer openen.

### Direct ZIP-download

Voor degenen die de Git-ervaring willen overslaan:

- Selecteer het vervolgkeuzemenu 'Code' rechtsboven en klik op Download ZIP.
- Pak het ZIP-bestand uit naar een map naar keuze.
- Voeg de Basis-submap toe als een bestaand project in Unity Hub.
