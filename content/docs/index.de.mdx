Willkommen im Basis-Wiki!

Diese Seite hilft Ihnen dabei, die minimalen Schritte zu unternehmen, um ein Basis-Projekt zum Laufen zu bringen, einschließlich Anleitungen zum Erstellen von Avatar- und World-Asset-Bundles.

## Aktuelle Systemanforderungen

https://docs.unity3d.com/Manual/system-requirements.html

## Unity installieren

- Hub - https://unity.com/download
- Standalone - https://unity.com/releases/editor/archive

### Kompatible Unity-Versionen

Bitte beachten Sie je nach verwendetem Branch die Version des Unity Editors, auf der das Framework basiert:

✓ [Developer](https://github.com/BasisVR/Basis/blob/developer/Basis/ProjectSettings/ProjectVersion.txt#L1)  
✓ [Long-Term-Support](https://github.com/BasisVR/Basis/blob/long-term-support/Basis/ProjectSettings/ProjectVersion.txt#L1)

### Zusätzliche Unity-Module

Fügen Sie beim Installieren der Unity Editor-Version die folgenden Module über den Bildschirm "Module hinzufügen" hinzu:

```ts tab="Windows"
✓ Android Build Support  
   ↳ ✓ OpenJDK  
   ↳ ✓ Android SDK & NDK Tools  
✓ IOS Build Support  
✓ Linux Build Support (Mono) 
✓ Mac Build Support (Mono)
✓ Windows Build Support (IL2CPP)
```

```ts tab="Mac"
✓ Android Build Support  
   ↳ ✓ OpenJDK  
   ↳ ✓ Android SDK & NDK Tools  
✓ IOS Build Support  
✓ Linux Build Support (Mono)
✓ Mac Build Support (IL2CPP)
✓ Windows Build Support (Mono)
```

```ts tab="Linux"
✓ Android Build Support  
   ↳ ✓ OpenJDK  
   ↳ ✓ Android SDK & NDK Tools  
✓ IOS Build Support  
✓ Linux Build Support (IL2CPP)
✓ Mac Build Support (Mono)
✓ Windows Build Support (Mono)
```

<Callout type="info">
Auch wenn Sie nur benutzergenerierte Inhalte für Avatar- und World-Erstellung erstellen, benötigen Sie diese zusätzlichen Module, um die Inhalte auf diesen Plattformen verfügbar zu machen.
</Callout>

<Callout type="info">
Wenn Sie bereits die richtige Unity-Version installiert haben, die Option "Module hinzufügen" jedoch nicht verfügbar ist, wurde sie wahrscheinlich außerhalb von Unity Hub installiert. Bitte installieren Sie die richtige Unity-Version über Unity Hub neu.
</Callout>

## Das Basis-Framework erwerben

### Git

Sie können die `developer`- oder `long-term-support`-Branches direkt [klonen](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository) unter https://github.com/BasisVR/Basis oder in Ihr eigenes lokales Repository [forken](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/working-with-forks/fork-a-repo) und dann das resultierende Unity-Projekt auf Ihrem Computer öffnen.

### Direkter ZIP-Download

Für diejenigen, die auf Git verzichten möchten:

- Wählen Sie das Dropdown-Menü "Code" oben rechts und dann ZIP herunterladen.
- Entpacken Sie die ZIP-Datei in einen beliebigen Ordner.
- Fügen Sie den Basis-Unterordner als vorhandenes Projekt in Unity Hub hinzu.
