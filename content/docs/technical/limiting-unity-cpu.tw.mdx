---
title: 限制 Unity CPU 使用率
---

在 Unity 6 上建置專案時，它往往會使用您系統的所有資源，即使在高階機器上也會導致系統完全無法使用（尤其是在著色器編譯階段）。

您可以將以下參數用作啟動器參數，以阻止其將每個 CPU 核心鎖定在 100%。

```
-job-worker-count 8
```

其中 **8** 應該是小於您機器實際物理 CPU 核心總數的數字。這將把 Unity 子工作程序限制在此值，從而防止其鎖定每個 CPU 核心。這甚至可能提高 Unity 的建置速度，因為它往往會過度競爭並使自身減速。
