## Een netwerk-gesynchroniseerde prop maken

Props kunnen netwerk-gesynchroniseerd zijn en tijdens uitvoering worden gegenereerd. Hieronder is een voorbeeld van hoe je een netwerk-gesynchroniseerde Pickup Interactable-kubus laadt:

- Voeg een kubusobject toe in de scènehiërarchie.

![Inspectorvenster bij Cube](/img/props/2A.png)

- Voeg een Basis Pickup Interactable-component toe. Hiermee kan een lokale speler de kubus oppakken en verplaatsen.

![Inspectorvenster bij Basis Pickup Interactable-component](/img/props/2B.png)

- Voeg een Basis Object Sync Networking-component toe. Hiermee wordt de kubus gesynchroniseerd voor alle spelers.

![Inspectorvenster bij Basic Object Sync Networking-component](/img/props/2C.png)
