## 概述

道具在 Basis 框架專案中提供了豐富的社交 VR 體驗動態。本文件將引導您使用 Unity 和 Basis SDK 建立、構建、上傳和自訂您自己的道具。

Basis 使用 `.BEE` 檔案格式儲存道具，將編譯後的資產和元資料打包成一個易於共享的單一檔案。道具支援混合形狀、材質和腳本。

## 主要功能

- **人形骨骼支援**：標準 Unity 人形骨骼，支援自動骨骼映射
- **混合形狀**：完整支援面部表情和口型同步
- **多平台**：支援 Windows、Linux、Mac、Android 和 iOS 建構
- **URP 著色器**：與 Unity 的通用渲染管線相容

## 需求

- **Unity**：Unity 6（最新支援版本）
- **模型格式**：帶人形骨骼的 FBX
- **著色器**：URP 相容著色器（Lit、Poiyomi、lilToon 等）
- **託管**：.BEE 檔案的遠端 URL 託管（Google Drive 最大 100MB）

## 工作流程概述

1. 將虛擬形象模型**匯入** Unity
2. **設定**骨骼和蒙皮網格
3. **新增** Basis Avatar 元件
4. 為目標平台**建構** .BEE 檔案
5. **上傳**到託管服務
6. 使用 URL 和密碼在 Basis 中**載入**
