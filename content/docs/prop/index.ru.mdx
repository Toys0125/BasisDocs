## Обзор

Пропы обеспечивают широкий спектр динамики социального VR-опыта в проекте на фреймворке Basis. Эта документация проведёт вас через создание, сборку, загрузку и настройку собственных пропов с помощью Unity и Basis SDK.

Basis использует формат файла `.BEE` для пропов, который упаковывает скомпилированные ресурсы и метаданные в единый удобный для обмена файл. Пропы поддерживают блендшейпы, материалы и скриптинг.

## Ключевые возможности

- **Поддержка гуманоидного рига**: Стандартные гуманоидные риги Unity с автоматическим маппингом костей
- **Блендшейпы**: Полная поддержка мимики и визем
- **Мультиплатформенность**: Сборка для Windows, Linux, Mac, Android и iOS
- **URP-шейдеры**: Совместимость с Universal Render Pipeline Unity

## Требования

- **Unity**: Unity 6 (последняя поддерживаемая версия)
- **Формат модели**: FBX с гуманоидным ригом
- **Шейдер**: URP-совместимые шейдеры (Lit, Poiyomi, lilToon и др.)
- **Хостинг**: Удалённый URL-хостинг для файлов .BEE (макс. 100 МБ для Google Drive)

## Обзор рабочего процесса

1. **Импортируйте** модель аватара в Unity
2. **Настройте** риг и скиннед-меши
3. **Добавьте** компонент Basis Avatar
4. **Соберите** файл .BEE для целевых платформ
5. **Загрузите** на хостинг-сервис
6. **Загрузите** в Basis, используя URL и пароль
