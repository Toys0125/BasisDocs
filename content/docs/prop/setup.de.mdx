## Den Unity Editor einrichten

- Gemäß dem Abschnitt [Erste Schritte](/en/docs#installing-unity) zum Installieren der neuesten unterstützten Unity-Editor-Version und zum Erwerb des Basis-Frameworks.
- Öffnen Sie Unity Hub und fügen Sie ein vorhandenes Projekt hinzu, wählen Sie den extrahierten/geklonten Basis-Projektordner
- Beim ersten Laden des Projekts erscheint die folgende Aufforderung, bitte klicken Sie auf 'Restart Editor'.

![Unity Neustart-Aufforderung](/img/avatars/0.png)

- Sobald das Projekt geladen ist, können Sie Ihre Avatar-Dateien laden:
- Entweder,
  - Finden Sie `Assets` in der Projekthierarchie, klicken Sie mit der rechten Maustaste auf einen leeren Bereich innerhalb von `Assets` und wählen Sie `Show in Explorer`
  - Kopieren Sie Ihre Avatar-Dateien [Mesh/Materialien/Texturen, usw.]
- oder
  - Importieren Sie ein .unity-Paket mit Ihren Dateien, über das Assets-Menü, Import Package > Custom Package
- Wechseln Sie zurück zum Unity-Fenster und lassen Sie den Importvorgang abschließen.

<Callout type="info">
Wenn Sie Ihren Inhalt in einem neuen Verzeichnis "Assets > _UserContent" ablegen, wird die Standard-.gitignore-Datei des Projekts Änderungen in diesem Ordner in git ausblenden.
</Callout>

## Modell importieren

- Finden Sie den Speicherort Ihres Prop-Modells (normalerweise im .FBX-Format bereitgestellt).
- Klicken Sie auf die Datei und navigieren Sie dann zum Abschnitt `Model` des Import Settings-Fensters.
- Stellen Sie sicher, dass `Red/Write` und `Legacy Blend Shape Normals` ausgewählt sind.

## Skinned Mesh Einrichtung

- Eine Überlegung für Props mit mehreren Skinned Meshes ist, wenn unterschiedliche Begrenzungsboxgrößen dazu führen, dass einzelne Meshes ein- und ausgeblendet werden, während der Rest des Props für entfernte Betrachter sichtbar ist.
- Sie können die Center X, Y, Z & Extent X, Y, Z-Werte im Abschnitt `Bounds` oben in der Skinned Mesh Renderer-Komponente überschreiben. Normalerweise ist es eine gute Idee, dies bei den separaten Skinned Meshes gleich zu setzen.

## Materialien/Shader

- Sie können direkt mit dem Universal Render Pipeline/Lit-Shader einsteigen, der mit Unity 6 mitgeliefert wird
- Andernfalls haben wir eine Liste kompatibler [Avatar-Shader](/de/docs/avatar/shaders) zusammengestellt.

## Basis Prop Einrichtung

- Die obigen Schritte sollten automatisch eine Animator-Komponente zu Ihrem Avatar-Prefab hinzufügen.
- Klicken Sie auf dem übergeordneten GameObject des Avatars auf `Add Component` und fügen Sie `Basis Prop` hinzu.
- Geben Sie Ihren bevorzugten Prop-Namen und eine Beschreibung ein.
- Bevor Sie `Create Prop Bee File` auswählen, stellen Sie sicher, dass alle Ihre Shader kompatibel sind. Die BEE-Datei wird nicht generiert, wenn es inkompatible Shader-Fehler auf Ihrem Prop gibt. Wenn es Probleme gibt, wird der Inspektor berichten — beheben Sie diese!
