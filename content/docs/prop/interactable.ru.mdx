## Как создать синхронизированный по сети проп

Пропы могут синхронизироваться по сети и генерироваться в рантайме. Ниже пример загрузки синхронизированного по сети куба Pickup Interactable:

- Добавьте объект куба в иерархию сцены.

![Окно инспектора на кубе](/img/props/2A.png)

- Добавьте компонент Basis Pickup Interactable. Это позволяет локальному игроку поднимать и перемещать куб.

![Окно инспектора на компоненте Basis Pickup Interactable](/img/props/2B.png)

- Добавьте компонент Basis Object Sync Networking. Это позволяет синхронизировать куб для всех игроков.

![Окно инспектора на компоненте Basic Object Sync Networking](/img/props/2C.png)
