## Descripción general

Los scripts proporcionan funcionalidad avanzada dentro del framework Basis. Esta documentación le guiará en la creación, compilación y prueba de scripts usando Unity y el SDK de Basis.

Basis también hace uso del paquete Unity `Cilbox`, un emulador CIL en sandbox.
