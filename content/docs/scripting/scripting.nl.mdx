## Scripts toevoegen via Unity Client Build

<Callout type="info">

Als je een game of applicatie bouwt met het framework en het ondersteunen van gebruikersgegenereerde content (zoals scènes, avatars of props) geen prioriteit is, geef je er misschien de voorkeur aan je MonoBehaviour-scripts native te bouwen in plaats van Cilbox te gebruiken.

Dit komt doordat Cilbox een CIL-interpreter is, die een aanzienlijke snelheidsstraf met zich meebrengt en mogelijk slechts een subset van beschikbare taaleigenschappen ondersteunt.
</Callout>

- Maak een canvas-object aan met een knop erin.

- Voeg een script toe aan het knopobject. Als eenvoudig voorbeeld maken we een script dat een TMP-tekstobject bijwerkt, genaamd TestButton.

- Maak een TMP-tekstobject aan en wijs het toe aan het veld Button Text in de inspector.

![Inspectorvenster met TestButton-component](/img/scripting/1.png)

public class TestButton : MonoBehaviour

    [SerializeField] private TextMeshPro buttonText;

    public void OnButtonClick()

## Netwerk-synchronisatiescripts toevoegen via Unity Client Build

- Maak een canvas-object aan met een knop erin.

![Inspectorvenster met canvas](/img/scripting/9.png)

- Voeg een script toe aan het knopobject. Als eenvoudig voorbeeld maken we een script genaamd SCNE (afkorting voor Send Custom Network Events) dat een TMP-tekstobject bijwerkt met het aantal keren dat een knop wordt geklikt.

- Maak een TMP-tekstobject aan en wijs het toe aan het veld Button Text in de inspector.

- Selecteer de knop en voeg er een onClick()-event aan toe. Wijs je Button Gameobject eraan toe en zorg ervoor dat je SCNE.IncrementCounter selecteert als de methode die je knop aanroept wanneer erop wordt geklikt.

``` cs title="SCNE.cs"
using Basis;
using Basis.Network.Core;
using Basis.Scripts.Networking.NetworkedAvatar;
using System;
using TMPro;
using UnityEngine;

public class SCNE : BasisNetworkBehaviour
{
    private int counter = 0;

    [SerializeField] TextMeshPro counterText;

    public void IncrementCounter()
    {
        TakeOwnership(); // Ensure we have ownership before sending the update
        counter++;
        SendCustomNetworkEvent(BitConverter.GetBytes(counter), DeliveryMethod.ReliableOrdered);
        OnDeserialization();
    }

    public override void OnNetworkMessage(ushort playerId, byte[] buffer, DeliveryMethod deliveryMethod) // Called when a network message is received
    {
        if (buffer != null && buffer.Length >= 4)
        {
            counter = BitConverter.ToInt32(buffer, 0);
            OnDeserialization();
        }
    }

    public void OnDeserialization() // Called after any change (local or remote)
    {
        Debug.Log($"Counter value is now: {counter}");
        counterText.text = $"Counter: {counter}";
    }

    public override void OnPlayerJoined(BasisNetworkPlayer player) // Ensure late joining players receive the correct count
    {
        if (IsLocalOwner())
        {
            SendCustomNetworkEvent(BitConverter.GetBytes(counter), DeliveryMethod.ReliableOrdered, new ushort[] { player.playerId }); // Send the current counter value to the new player
        }
    }
}
```

![Inspectorvenster met knop met SCNE-script](/img/scripting/10.png)

<Callout type="info">
De overschreven **OnPlayerJoined()** wordt gebruikt omdat, zonder deze, nieuw toetredende spelers anders het bijgewerkte telnummer niet zouden zien.

**TakeOwnership()** wordt gebruikt om desynchronisatie te voorkomen, zoals het resetten van de tellerwaarde doordat een nieuw toetredende speler op de knop klikt.
</Callout>

## Netwerk-synchronisatiescripts toevoegen via Cilbox

Cilbox is een scriptsysteem dat als lijm dient om minimale spellogica te bieden, waardoor basis gedragingen in gebruikersgegenereerde content zoals props, avatars en scènes overdraagbaar worden. Het kan worden gebruikt om scripts te maken die tijdens uitvoering werken.

Momenteel is Cilbox nog erg experimenteel. Het volgende voorbeeld laat zien hoe je een script maakt om een kubus te laten roteren met Cilbox.

<Callout type="info">
Opmerking: Cilbox is met name nuttig als je een applicatie of game maakt die gebruikersgegenereerde content laadt en je scripting op die gebruikerscontent wilt ondersteunen. Het maakt het mogelijk scripts te verzenden als iets als DLC naast je andere content, terwijl het ook een veiligere sandbox biedt voor het uitvoeren van code, waardoor scripts met schadelijk of ongewenst gedrag worden voorkomen.
</Callout>

- Maak een kubusobject aan en een ObjectRotator-component om eraan te koppelen. Zorg ervoor dat je [Cilboxable] boven je klasse toevoegt.

```cs title="ObjectRotator.cs"
using UnityEngine;

[Cilboxable]
public class ObjectRotator : MonoBehaviour
{
    [SerializeField] private Vector3 rotationSpeed = new Vector3(0, 100f, 0);

    private void Update()
    {
        transform.Rotate(rotationSpeed * Time.deltaTime);
    }
}
```

Hieronder staan enkele aanvullende voorbeeldscripts die je kunt uitproberen.

``` cs title="NM_Wind "
using UnityEngine;

[Cilboxable]
public class NM_Wind : MonoBehaviour
{
    public Texture2D NoiseTexture;
    public Texture2D GustMaskTexture;
    public void Start()
    {
        ApplySettings();
    }
    void ApplySettings()
    {
        Shader.SetGlobalTexture("WIND_SETTINGS_TexNoise", NoiseTexture);
        Shader.SetGlobalTexture("WIND_SETTINGS_TexGust", GustMaskTexture);
        Shader.SetGlobalVector("WIND_SETTINGS_WorldDirectionAndSpeed", new Vector4(1, 1, 1, 30 * 0.2777f));
        Shader.SetGlobalFloat("WIND_SETTINGS_FlexNoiseScale", 1.0f / 175.0f);
        Shader.SetGlobalFloat("WIND_SETTINGS_ShiverNoiseScale", 1.0f / 10.0f);
        Shader.SetGlobalFloat("WIND_SETTINGS_Turbulence", 30 * 0.25f);
        Shader.SetGlobalFloat("WIND_SETTINGS_GustSpeed", 50);
        Shader.SetGlobalFloat("WIND_SETTINGS_GustScale", 1.0f);
        Shader.SetGlobalFloat("WIND_SETTINGS_GustWorldScale", 0.0016666667f);
    }
}
```

```cs title="Gun"
using UnityEngine;
using Basis.Scripts.BasisSdk.Interactions;
using Basis.Scripts.Device_Management.Devices;

[Cilboxable]
public class Gun : MonoBehaviour
{
    private BasisPickupInteractable pickup;

    void Start()
    {
        this.pickup = GetComponent<BasisPickupInteractable>();
        this.pickup.OnInteractStartEvent.AddListener(OnPickup);
        this.pickup.OnInteractEndEvent.AddListener(OnDrop);
        this.pickup.OnPickupUse.AddListener(OnUse);
    }

    private void OnPickup(BasisInput input)
    {
        Debug.Log("HoloGun: Picked up!");
    }

    private void OnDrop(BasisInput input)
    {
        Debug.Log("HoloGun: Dropped!");
    }

    void OnUse(BasisPickUpUseMode mode)
    {
        switch (mode)
        {
            case BasisPickUpUseMode.OnPickUpUseDown:
                Debug.Log("HoloGun: Pew Pew!");
                break;
            case BasisPickUpUseMode.OnPickUpUseUp:
                Debug.Log("HoloGun: Pew Pew! (released)");
                break;
        }
    }
}
```

In plaats van BasisNetworkBehaviour kan BasisNetworkShim worden overgeërfd om genetwerkte scripts te maken.
</Callout>
