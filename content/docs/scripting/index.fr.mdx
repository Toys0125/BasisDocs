## Vue d'ensemble

Les scripts fournissent des fonctionnalités avancées dans le framework Basis. Cette documentation vous guidera dans la création, la compilation et le test de scripts avec Unity et le SDK Basis.

Basis utilise également le package Unity `Cilbox`, un émulateur CIL en sandbox.
